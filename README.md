## Banking Transaction Analysis, Data Cleaning & MIS Dashboard using Advanced Excel

* Developed a banking transaction analysis project using Advanced Excel.
* Performed data cleaning and preprocessing by removing duplicates, handling missing values, and standardizing customer records.
* Created Pivot Tables, MIS reports, and interactive dashboards for transaction analysis.
* Built branch-wise transaction reports using charts, slicers, and conditional formatting.
* Used Excel formulas such as IF, SUMIF, VLOOKUP, TRIM, and PROPER for data processing and automation.
* Designed KPI dashboards to track total deposits, withdrawals, and transactions.
